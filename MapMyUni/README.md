## 📖 Overview
**MapMyUni** is a full-stack web application developed to simplify campus navigation at Christ University Delhi NCR. Built with a focus on accuracy and utility, the platform serves as a digital directory for rooms, departments, and faculty members.

### 👣 The "Behind the Scenes" Grind
What sets this project apart is the **Primary Fieldwork**. For over a month, our team physically mapped every corner of the campus:
*   **Manual Verification:** We walked every block, floor, and room to ensure the digital map matches the real world.
*   **Data Integrity:** No shortcuts or borrowed info. We collected faculty details and location data ourselves to ensure 100% reliability for our peers.
*   **From Local to Live:** Transitioned from a local development environment to a fully hosted web solution.

---

## 🚀 Core Features
*   **Interactive Campus Locator:** Find any block, department, or specific room number instantly.
*   **Faculty Directory:** A comprehensive, SQL-powered listing of faculty organized by department.
*   **Visual Campus Gallery:** A high-quality visual look at the university's infrastructure.
*   **Dynamic UI Experience:** Includes a custom **Light/Dark Mode** toggle for optimal user experience.
*   **Responsive Design:** Fully optimized for mobile devices so students can navigate while walking.
